GetFolderMessages
=================

安全性の低いアプリからのアカウントへのアクセスを許可する必要があります。

- [安全性の低いアプリからのアカウントへのアクセスを許可する - Google アカウント ヘルプ](https://support.google.com/accounts/answer/6010255)


動作確認環境
------------

Java: openjdk version "11" 2018-09-25


実行
----

```
gradle run
```
