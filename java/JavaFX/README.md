JavaFX
======

JavaFX の勉強です。


Usage:
------

環境変数 `ORG_GRADLE_PROJECT_JAVA_HOME` に、 Java のインストールパスを設定してください。

```sh
export JAVA_HOME=/c/Java/jdk1.8.0_131
export ORG_GRADLE_PROJECT_JAVA_HOME=$JAVA_HOME
```

各プログラムの実行方法は下記の通りです。

```sh
# s01_HelloWorld
java -cp "build/libs/JavaFX.jar" jp.dip.oyasirazu.study.javafx.HelloWorld

# s02_ButtonAndLabel
java -cp "build/libs/JavaFX.jar" jp.dip.oyasirazu.study.javafx.ButtonAndLabel

# s03_ListView
java -cp "build/libs/JavaFX.jar" jp.dip.oyasirazu.study.javafx.ListView

# s04_ListViewAddItem

# s06_AddPanel
```
